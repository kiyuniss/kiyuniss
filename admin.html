How to use Google Drive photo:
  1. Click Open Google Drive above
  2. Upload your photo to Drive
  3. Right click photo → Share → Anyone with link
  4. Right click photo → Get link → Copy it
  5. The link looks like: https://drive.google.com/file/d/FILEID/view
  6. Copy just the FILEID part
  7. Paste this in the URL field: https://drive.google.com/uc?export=view&id=FILEID

Or use imgbb — upload photo, copy Direct link, paste above.

Library Management

How to add PDF study materials to the student library

How to Upload PDFs

  1. Open your kiyuniss.github.io folder in VS Code
  2. O Level PDFs go in: library/o-level/[subject]/
  3. A Level PDFs go in: library/a-level/[subject]/
  4. Name files clearly: e.g. physics-notes-waves.pdf
  5. Open library.html and change path:"#" to the file path
  6. Save then run: git add . && git commit -m "added pdf" && git push
